## Installation

Setting up PostgreSQL on Laravel

- psql
- create database laravel;
- create user laravel with encrypted password 'laravel';
- grant all privileges on database laravel to laravel;
- /q

Congifure .env file:
- DB_CONNECTION=pgsql
- DB_HOST=127.0.0.1
- DB_PORT=5432
- DB_DATABASE=laravel
- DB_USERNAME=laravel
- DB_PASSWORD=laravel
